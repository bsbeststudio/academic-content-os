# Predoc 申请前 Checklist

---

## CV 准备

- [ ] Research Experience 占比合理（40–60%）
- [ ] 每段研究经历描述包含：数据来源、方法工具、主要任务、输出物
- [ ] 所有列出的技能可在面试中演示
- [ ] Replication 经历（如有）标注清楚，描述包含原论文和与原文的异同
- [ ] 篇幅控制在 1–2 页

## Cover Letter 准备

- [ ] 已针对目标 PI 的具体论文/项目做定制
- [ ] Cover Letter 以 PI 项目为中心，而非候选人收益为中心
- [ ] 最强经历已在开头突出，未被埋没在平行列表中
- [ ] 篇幅控制在 1 页

## 推荐信准备

- [ ] 推荐人列表确认（写信人姓名、职位、合作性质）
- [ ] 每位写信人：在科研情境下直接观察过你？（否则重新考虑）
- [ ] 三封信组合覆盖不同维度（非同一机构/同一关系）
- [ ] 已提前告知写信人申请时间线和目标程序

## 目标 PI 研究

- [ ] 了解目标 PI 最近 2–3 篇论文内容
- [ ] 能说出目标 PI 用了什么识别策略
- [ ] 已找到自己经历与 PI 研究的具体交叉点

## 签证/资格确认

- [ ] 确认目标职位的工作资格要求（OPT/H1B/其他）
- [ ] 确认 OPT 有效期是否覆盖目标 Predoc 时长
- [ ] 如 OPT 即将到期，已评估 STEM 延期资格
