# 面试前 24 小时清单

---

## 内容准备

- [ ] 读目标 PI 最近 1–2 篇论文，找出与自己研究兴趣的具体连接点（一句话概括）。
- [ ] 准备 3 个问题问面试官，至少一个关于具体在研项目。
- [ ] 确认签证/时间线细节，做到对答如流，无需停顿。

## 口头表达练习

- [ ] 大声朗读研究兴趣叙事 3 遍，确保听起来像第一次想到的，而不是在背。
- [ ] 计时练习 replication walkthrough，目标 90 秒内。
- [ ] 计时练习 robustness test 解释，目标 90 秒内。
- [ ] 练习 AI "初级 RA" 比喻，做到无停顿流畅输出。

## 心理准备

- [ ] 回顾红线清单，识别自己个人风险最高的一条。
- [ ] 提醒自己：面试官的委婉批评是调整信号，不是对话邀请——听到后立即重构，不要辩护。
- [ ] 提醒自己："I don't know" 是可接受的。试图掩盖不知道才是真正的问题。
- [ ] 提醒自己：面试官主动追问是好信号，不要慌张，不要急着抢话。
