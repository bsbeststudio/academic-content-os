# 推荐信准备 Checklist

---

## 写信人评估

- [ ] 写信人是否在北美终身轨教授层级？（最优先）
- [ ] 写信人是否在科研情境下直接观察过我？（PI-RA 关系 > 课程）
- [ ] 写信人在北美学术网络中是否有直接联系？（可核实性）
- [ ] 三封信是否覆盖不同维度（科研、课程、其他）？

## 内容沟通

- [ ] 是否已向写信人提供了最新 CV？
- [ ] 是否已向写信人提供了 Cover Letter / SOP 草稿？
- [ ] 是否已说明申请程序的具体研究方向，帮助写信人针对性描述？
- [ ] 是否已说明申请截止日期和提交方式？

## 时间线管理

- [ ] 是否在申请截止前 6–8 周联系写信人？
- [ ] 是否已在截止前 2 周发提醒？
- [ ] 是否有备选写信人（如某位写信人临时无法提交）？
