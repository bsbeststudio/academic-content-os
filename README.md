# Academic Content OS

这是我的 Academic Career 内容知识库。

## 目标

维护所有知识，而不是维护文章。

文章由 AI 根据知识自动生成。

## Hub

Academic Careers

Graduate School

Research Skills

Career Development

## 每个 Topic 包括

Knowledge

Opinion

Framework

Coaching

Decision Rule

FAQ

Golden Sentence

Checklist

Playbook

## Repository Structure

```
taxonomy/        # topic & hub definitions
content/
  knowledge/     # objective information
  frameworks/    # reusable structures
  opinions/      # professional judgment
  coaching/      # teaching experience & common mistakes
assets/
  decision_rules/  # IF → THEN logic
  checklists/      # step-by-step preparation
  faq/             # frequently asked questions
  playbooks/       # multi-step workflows (SOPs)
  golden_sentences/ # reusable phrases
generated/       # AI-generated articles (do not edit directly)
```

## 原则

Knowledge 必须客观。

Opinion 必须来自经验。

Framework 必须可执行。

任何文章都不直接编辑，而是由 Knowledge 自动生成。
