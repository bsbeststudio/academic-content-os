# 面试前准备 Checklist（通用）

---

## 内容准备

- [ ] 2分钟自我介绍准备完毕（突出最强研究经历）
- [ ] 5分钟自我介绍准备完毕（完整研究叙事）
- [ ] 每段简历研究经历：能流利说出数据来源、方法、输出
- [ ] 读过目标 PI 最近 2–3 篇论文，能说出具体内容
- [ ] Coding Exercise 所有技术选择能口头解释（含具体 bug）

## 答题框架

- [ ] 研究兴趣叙事：能在 3 分钟内说清主线
- [ ] 兴趣转变：能用"自然延伸"框架解释
- [ ] 背景匹配：能承认差距 + 转化为互补性 + 点名学习计划
- [ ] 因果推断：如无干净识别策略，能诚实承认 + 提出改进
- [ ] 职业规划：目标不局限于单一地区/机构

## Closing Question

- [ ] 准备 2 个高价值 Closing Question（非 JD 中已有内容）

## 行为问题

- [ ] 分歧处理：准备一个真实情境（不是假设）
- [ ] 时间管理/优先级：准备一个具体例子
